Small application for image browsing.

Environmet to build : Windows 10, Visual Studio 2017

Libraries/packages used in solution : VirtualizingWrapPanel , Moq

## Supported functionality

Supported image extensions : .BMP,.GIF,.EXIF,.JPG,.PNG,.TIFF

Images loading by drag&drop single file, multiple files or folder

Open single photo mode by clicking on image in Gallery

Switch to next/previous image with left/right arrows and left/right keys

Return to Gallery Mode with ESC key
